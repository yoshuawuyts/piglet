# piglet


A tiny grunt configuration

## Tasks

__CSS:__ Styl, Autoprefixer, CSSo optimizer

__Views:__ Jade

__JS:__ Concat, Uglify

__Lint:__ JSbeautifier, JShint

With autoreload, watch and more...
